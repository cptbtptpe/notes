### 按小时分组统计 ($hour, $month, $week, ...)  

```  
db.device.aggregate([  
    {  
        $group: {  
            _id: {$hour: "$addtime"},  
            count: {$sum: 1}  
        }  
    },  
    {$sort: {_id: 1}}  
])  
```  
  
### 日期区间查询

```  
db.log_register.find({  
    addtime: {  
        $gt: new Date(2016,4,13,23),  
        $lt: new Date(2016,4,14,8)  
    },  
    type: 1  
}).count()  
```
